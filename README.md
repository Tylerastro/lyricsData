# 歌詞數據

此專案包含了歌詞資料，我們架設網站在 [https://lyrics-zeta.vercel.app](https://lyrics-zeta.vercel.app) 上提供歌詞學習的體驗。

## 關於

這個開源專案作為歌詞資料庫使用。這裡儲存的資料用於網站與影片一起展示歌詞，提升音樂聆聽和歌詞沈浸體驗。

## 網站
